# **App Name**: Kakee: Fluid Glamour

## Core Features:

- Home Screen Showcase: Display user's 'General' rank status with animated profile frame, Koin balance, personalized content, and latest content drops.
- Music Discovery: Enable fluid music discovery with dynamic headers, curated playlists, and 'Now Playing' bar.
- Vertical Video Immersion: Offer immersive short-form video experience with full-screen player, interactive elements, and comment panel.
- Rewarding Engagement: Integrate 'Daily Spin,' Koin balance display, exclusive deals ('Lobangs'), and mini-games arcade to incentivize user engagement.
- Personalized Recommendations: Employ generative AI as a tool that uses viewing and listening history to recommend 'Recently for You' content on the Home tab.
- Trend Prediction: Leverage machine learning algorithms to predict trending content in the 'Listen' and 'Watch' tabs, ensuring users always discover fresh and engaging material.
- Smart Search: Implement an intelligent search functionality in the 'Watch' tab that offers relevant content suggestions and genre-based browsing, enhancing content discovery.

## Style Guidelines:

- Primary color: Vibrant yellow (#FFDA63) to highlight active elements and key information.
- Background color: Very light, desaturated yellow (#F2EFC4) to maintain a bright and airy feel.
- Accent color: Soft amber (#FFB347) to emphasize CTAs and add warmth.
- Body and headline font: 'Poppins' (sans-serif) for a modern, precise look.
- Use vibrant, custom icons that align with Singaporean culture and media content.
- Employ fluid, non-scrolling layouts for static tabs, ensuring all content fits within a single screen. Dynamic, shrinking headers should be used in scrollable tabs like 'Listen.'
- Incorporate subtle animations and transitions throughout the app to enhance the feeling of fluidity and glamour. This includes dynamic scrolling headers, radiant glows, and smooth panel transitions.
